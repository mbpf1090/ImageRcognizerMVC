# Image Recognizer

## Setup
Enter your keys for Azure Cognitive/Speech Services in User Secrets:
```
{
  "Vision": {
    "subscriptionKey": "<YOUR-VISION_KEY>",
    "endpoint": "<YOUR-VISION-ENDPOINT-URL"
  },
  "Speech": {
    "subscriptionKey": "<YOUR-SPEECH-KEY",
    "location": "><YOUR-LOCATION>"
  }
}
```
